Política de Privacidad de Bijou Track

Última actualización: 15/04/2026

Bijou Track es una aplicación de inventario de bisutería diseñada para funcionar principalmente de forma local en el dispositivo del usuario.

1. Información que utiliza la aplicación

Bijou Track puede acceder a la cámara y a imágenes o archivos del dispositivo para escanear códigos QR, buscar piezas por imagen, seleccionar fotos, personalizar el logo y exportar o importar copias de seguridad.

2. Cómo se usa la información

La información se utiliza únicamente para gestionar piezas, cajas y ubicaciones, escanear códigos QR, buscar piezas por imagen, personalizar la aplicación y crear o restaurar copias de seguridad.

3. Almacenamiento de datos

Los datos del inventario se almacenan localmente en el dispositivo del usuario. Salvo que el usuario exporte manualmente una copia de seguridad o comparta un archivo, la información no se envía a servidores externos.

4. Compartición de datos

Bijou Track no vende ni comparte datos personales con terceros con fines comerciales. Los datos solo podrán salir del dispositivo si el usuario decide exportarlos o compartirlos manualmente.

5. Seguridad

La aplicación procura tratar la información de forma segura dentro del dispositivo. El usuario es responsable de proteger el acceso a su teléfono y a los archivos exportados.

6. Conservación y eliminación de datos

Los datos permanecen almacenados en el dispositivo hasta que el usuario los elimine, desinstale la aplicación o importe una copia de seguridad que reemplace la información existente.

7. Permisos

Bijou Track solicita únicamente los permisos necesarios para su funcionamiento, como acceso a la cámara y a imágenes o archivos del dispositivo, cuando corresponde.

8. Menores de edad

Bijou Track no está dirigida específicamente a menores de 13 años.

9. Contacto

Si tienes preguntas sobre esta Política de Privacidad, puedes escribir a: tuemail@correo.com

10. Cambios en esta política

Esta Política de Privacidad puede actualizarse ocasionalmente. Cualquier cambio se publicará en esta misma página.
